# 🏠 루프
### 기획배경 및 설명

MZ 세대 사이 ‘갓생’ 문화가 확산되며 자기계발과 루틴 실천이 일상화되고 있습니다. 그러나 완벽주의적 루틴 압박과 비교 심리로 인해 오히려 스트레스와 번아웃을 경험하는 사례가 많습니다. 루프는 이러한 부담을 완화하고, 사용자가 **자신의 속도에 맞춰 지속 가능한 루틴을 형성할 수 있도록 돕는 서비스**입니다.

- 카카오 로그인 기반의 편리한 서비스 진입
- 목표별 루틴 생성 및 관리
- 매일 실천 내역 기록 및 시각적 확인

단순 체크리스트 앱이 아닌 사용자가 부담 없이 루틴을 유지하고 자기 효능감을 높일 수 있는 서비스가 되도록 구현했습니다.

- **개발 기간**: 2025.05 (2주)  
- **팀 구성**: PM 1명, DE 1명 FE 2명, BE 2명

### 나의 역할 및 구현 내용

- 카카오 소셜 로그인 연동 및 사용자 인증
- 사용자 목표 기반 루틴 추천 기능 구현
- 루틴 생성 및 조회 중심의  CRUD API 구현
- 응답 구조 통일화
